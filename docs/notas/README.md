# Notas de reunión

Minutas del equipo: acuerdos y pendientes con dueño y fecha. Una reunión por
archivo, nombrado `AAAA-MM-DD-tema.md`, usando la
[plantilla](plantilla-reunion.md).

Las decisiones que salgan de una reunión no viven aquí: los cambios de
alcance van al [control de cambios](../cambios/README.md) y las decisiones
técnicas a [../decisiones/](../decisiones/README.md); la minuta las enlaza.
